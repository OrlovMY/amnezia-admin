# amnezia-admin

Утилита администрирования сервера [Amnezia VPN](https://amnezia.org/) (AmneziaWG / WireGuard) через SSH.
Работает по административному ключу `vpn://…`, внутри которого — SSH-доступ к серверу.
Есть графическая версия (GUI на [Fyne](https://fyne.io/)) и консольная (CLI).

## Возможности

- Список пользователей: имя, дата создания, последняя активность, объём трафика.
- Сортировка по любой колонке (с запоминанием выбора).
- Создание пользователя с выдачей конфига и QR-кода.
- Переименование, временное отключение/включение без удаления, удаление.
- Перевыпуск конфига (re-key) — на случай утери или смены устройства.
- Локальное шифрованное хранилище ключей `vpn://` с доступом по пин-коду
  (Argon2id + XChaCha20-Poly1305, опциональная привязка к учётной записи Windows через DPAPI).

## Сборка

Требуется Go 1.21+. Для GUI нужен C-компилятор (CGO) — например MinGW-w64 на Windows.

```sh
# CLI (без CGO)
go build -o amnezia-admin.exe ./cmd/cli

# GUI (нужен CGO + gcc в PATH)
CGO_ENABLED=1 go build -ldflags "-s -w -H windowsgui" -o amnezia-admin-gui.exe ./cmd/gui
```

## Использование

Запустите GUI (`amnezia-admin-gui.exe`) и вставьте административный ключ `vpn://…`,
либо CLI:

```sh
amnezia-admin.exe list   -key vpn://...
amnezia-admin.exe add    -key vpn://... -name "Иван Иванов"
amnezia-admin.exe rename -key vpn://... -name "Иван Иванов" -newname "Иван Петров"
amnezia-admin.exe toggle -key vpn://... -name "Иван Иванов"
amnezia-admin.exe rekey  -key vpn://... -name "Иван Иванов"
amnezia-admin.exe del    -key vpn://... -name "Иван Иванов"
```

Ключ можно передать через переменную окружения `AMNEZIA_KEY`.

## Безопасность

Административный ключ `vpn://…` даёт полный SSH-доступ к серверу — храните его как секрет.
Локальное хранилище защищает ключ пин-кодом; для утекшего файла реальную стойкость
обеспечивают длина пин-кода и параметры Argon2id, а привязка к учётной записи Windows (DPAPI)
делает файл бесполезным на другом компьютере. Управление пользователями поддержано для
протоколов семейства WireGuard/AmneziaWG.
